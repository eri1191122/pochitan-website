# ポチ単 - 英検5級単語帳アプリ

英検5級の全579語を12種類のゲームで楽しく学べる完全無料iPhoneアプリ。

## 公式サイト

- **メインサイト**: https://pochitan-eiken.netlify.app/
- **App Store**: https://apps.apple.com/jp/app/id6757858287

## 特徴

- 579語すべてに7ステップ詳細解説
- 12種類のゲームで飽きずに学習
- 忘却曲線に基づく復習システム
- 短文穴埋め310問で実戦対策
- 保護者ダッシュボード搭載
- 完全無料・広告なし・課金なし・個人情報収集なし

## 対象

小学生・中学生の英検5級対策に最適です。
